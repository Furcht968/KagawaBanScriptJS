# 香川規制プログラム
## 香川規制プログラムとは？

香川県っぽいところからアクセスすると規制するプログラムです。

## 使い方(JS-CSS)

scriptを適用してBAN表示したい要素のタグ（基本的にdivやarticleとか）のクラスに"kagawa"を付けて解決ゥ！！
