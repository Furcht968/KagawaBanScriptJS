香川規制プログラムの説明書

下準備

jsファイルを読み込んでおいてください

例1 普通に表示する

記事タグなどのclassに"kagawa"を付けましょう。
ソースコード: 


<div class="kagawa">
</div>
<script src="./KBP.js">
</script>
出力（香川）：
条例によって禁止されています
香川県民はネット・ゲーム依存対策条例によりこのサービスの利用は規制されています。

例2　カスタマイズする

KBP.cssをロードし、"kagawa2"を用いた別のタグを使用します

ソースコード：
〜略〜
<link rel="stylesheet" href="./KBP.css">
</head>
<body>
 <div class="kagawa2">
  <h1>ネットはいいからさっさとうどん食え<h1>
 </div>
 <div class="kagawa">
 </div>
<body>
出力（香川）：
ネットはいいからさっさとうどん食え
使用はしないと思いますが作らせていただきました。
